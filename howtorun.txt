HOW TO RUN SADBHAVNATEA PROJECT IN CMD

--- Terminal 1: Backend ---
1. Open Command Prompt (cmd)
2. Run command: cd /d D:\SADBHAVNATEA\server
3. Run command: npm run dev

--- Terminal 2: Frontend ---
1. Open a new Command Prompt (cmd) window
2. Run command: cd /d D:\SADBHAVNATEA\client
3. Run command: npm run dev
