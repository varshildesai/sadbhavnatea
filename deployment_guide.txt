# SADBHAVNA TEA - DEPLOYMENT & PRODUCTION GUIDE

This guide provides step-by-step instructions on how to take your local e-commerce project and publish it to the internet for free, connect your admin panel, get listed on Google, and secure your authentication.

---

## 1. FREE HOSTING FOR THE BACKEND (Node.js & MongoDB)

To make your admin panel and database accessible worldwide, you need to host your Node/Express backend.

**A. Database Setup (MongoDB Atlas - FREE)**
1. Go to **mongodb.com/atlas** and create a free account.
2. Build a "Shared" (Free) Cluster.
3. Under "Database Access", create a Database User and save the password.
4. Under "Network Access", add IP Address `0.0.0.0/0` (Allows access from anywhere).
5. Click "Connect" -> "Drivers" to get your MongoDB Connection String. It looks like: 
   `mongodb+srv://<username>:<password>@cluster0.exmple.mongodb.net/sadbhavnatea?retryWrites=true&w=majority`

**B. Backend Hosting (Render.com - FREE)**
1. Create a free account on **Render.com**.
2. Connect your GitHub account and create a new "Web Service".
3. Select your GitHub repository for Sadbhavna Tea.
4. Set the Root Directory to `server`.
5. Build Command: `npm install`
6. Start Command: `node server.js`
7. Expand "Advanced" and add your Environment Variables (.env):
   - `MONGO_URI` = (Your MongoDB Atlas connection string from step A)
   - `JWT_SECRET` = (Create a long, random secure text string)
   - `NODE_ENV` = `production`
   - `PORT` = `5000`
8. Click Create. Render will give you a public URL (e.g., `https://sadbhavna-api.onrender.com`).

---

## 2. FREE HOSTING FOR THE FRONTEND (React/Vite)

**A. Update API URLs**
Before uploading the frontend, you must tell it to talk to the new live backend instead of localhost.
1. In your `client` folder, open `.env` (create it if it doesn't exist).
2. Add: `VITE_API_BASE_URL=https://sadbhavna-api.onrender.com` (Use your actual Render URL).
3. Search through your frontend code for `http://localhost:5000` and replace it with your newly hosted Render server URL.

**B. Frontend Hosting (Vercel.com - FREE)**
1. Go to **Vercel.com** and create a free account with GitHub.
2. Click "Add New" -> "Project".
3. Import your GitHub repository.
4. Set the Framework Preset to "Vite".
5. Set the Root Directory to `client`.
6. Click Deploy. 
7. Vercel will give you a fast, free live domain (e.g., `https://sadbhavnatea.vercel.app`).

---

## 3. GETTING A FREE DOMAIN

While Vercel gives you a `.vercel.app` domain, if you want a custom free domain:
1. Go to **Freenom.com** or **InfinityFree** (or buy a `.com` from Hostinger/Godaddy for very cheap).
2. If you buy/get a domain (e.g., `sadbhavnatea.com`), go to your Vercel Project Settings -> Domains.
3. Add your custom domain to Vercel. 
4. Vercel will give you DNS Records. Copy these to your domain provider's DNS settings.

---

## 4. CONNECTING THE ADMIN PANEL & REAL AUTHENTICATION

**A. Real Authentication (JWT Security)**
To ensure real security in production:
1. **Strong JWT Secrets:** Make sure your `JWT_SECRET` in the backend environment variables is a complex 64-character string.
2. **CORS Configuration:** In your `server/server.js`, you must restrict CORS so only your frontend can access the API:
   ```javascript
   app.use(cors({
     origin: ['https://sadbhavnatea.vercel.app', 'https://yourcustomdomain.com'],
     credentials: true
   }));
   ```
3. **Admin Users Setup:**
   Right now, you might have local admin users. In production, connect to your live MongoDB with a database viewer tool (like MongoDB Compass). Create your permanent Admin user account directly in the database, setting `isAdmin: true`.

---

## 5. UPLOADING TO GOOGLE & SEARCH ENGINES (SEO)

To make your site show up "all over browsers" and Google:

1. **Google Search Console (Crucial)**
   - Go to `search.google.com/search-console`.
   - Complete domain verification (Vercel has built-in ways to verify this).
   - This tells Google your site exists.

2. **Generate a Sitemap**
   - Since your data is dynamic, use an online tool or Vite plugins to generate a `sitemap.xml`.
   - Submit this `sitemap.xml` in the Google Search Console. Google will crawl your tea products automatically.

3. **Add Meta Tags to index.html**
   - In your `client/index.html`, add powerful SEO tags inside the `<head>`:
   ```html
   <title>Sadbhavna Tea | Premium Authentic Assam Tea</title>
   <meta name="description" content="Buy the finest Sadbhavna CTC and dust tea. We also provide Atlantis vending machines for your office.">
   <meta name="keywords" content="Assam Tea, Buy Tea Online, Vending Machines, Sadbhavna Tea, Tea Dust">
   ```

4. **Register on Google My Business**
   - If Sadbhavna Tea has a physical location or headquarters, list it on "Google Business Profile". This will put your brand on Google Maps and search sidebars.
