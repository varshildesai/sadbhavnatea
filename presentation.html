2. Introduction

  • Internship Overview: Practical implementation of full-stack software engineering concepts in a real-world e-commerce scenario.
  • Domain: Software Development (Web Applications)
  • Duration: 120 Hours

3. Project Overview

  • Project Name: Sadbhavna Tea E-Commerce Platform
  • Purpose of the Software: To provide a robust, fast, and secure online platform for users to browse, order, and track premium tea products seamlessly.
  • Target Users: Tea enthusiasts, wholesale buyers, and everyday consumers.

4. Scope of the Project

Features Included:

  • Dynamic Product Catalog & Wishlist
  • Secure Shopping Cart & Checkout (Razorpay)
  • User Authentication (Email OTP & Google OAuth)
  • Real-time Order Tracking & Notifications (Email/WhatsApp)
  • Comprehensive Admin Dashboard

User Roles: Customer, Admin

Limitations: Restricted to online payments only; delivery restricted to specific geographic zones initially.
